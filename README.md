# Introduction

A game project targeting can run

* windows
* linux
* mac
* iphone
* android
* wasm
* ....

# Requirements

* Clang
* Clang-Format

## Windows

* msvc
* visual studio 2022

# How to build

* git clone
* cd

## Windows

* run scripts/download_prj_tools.bat
* run scripts/setup_win_env.bat
* build_win.bat

## Linux

* run scripts/download_prj_tools.sh
* run scripts/setup_linux_env.sh
* build_linux.sh
